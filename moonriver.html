Moon River

Created using HTML/CSS Blueprints within Brackets.

This project was created as part of a course on HTML programming in Brackets, along with free 2D spritesheets downloaded from OpenGameArt.org.

View the concept document for a detailed rundown of the game!




During the development of this browser-based game, I managed to gain and improve the following skills:

• Using pointers and references.

• Using physics colliders to handle hit/touch boxes.

• Implementing and controlling audio.

• Looping and tweaking animations.

• Setting win and lose conditions.
