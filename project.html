Moon River

Created using HTML/CSS Blueprints within Brackets.

This project was created as part of a course on HTML programming in Brackets, along with free 2D spritesheets downloaded from OpenGameArt.org. During the development of this browser-based game, I managed to gain and improve the following skills:

• Using pointers and references.

• Using physics colliders to handle hit/touch boxes.

• Implementing and controlling audio.

• Looping and tweaking animations.

• Setting win and lose conditions.

How to install:

Requirements:

Google Chrome and Brackets

After installing both programs, open the Brackets application, click on 'File' - 'Open Folder', locate the folder containing the game files, and after it all loads, look for and select 'index.html'. You can then launch the page by clicking the lightning symbol on the upper right-hand side. This will put you into my game.
